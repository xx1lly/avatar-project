как запустить наш проект (примитивно) 

способ 1 - live server в vs code (рекомендую)

открой папку проекта в vs code
установи расширение Live Server
правой кнопкой по index.html → "Open with Live Server"
сайт откроется на http://127.0.0.1:5500


способ 2 - просто открыть файл
просто открой index.html в браузере — двойной клик или перетащи в окно браузера

способ 3 — через терминал (если есть node.js)

npx serve .
npx live-server


наш стек: vanilla js, css, html.
